Welcome to CARE Clinic

A one-time setup. Complete the three required checks, then Install & Start.

1 · Docker required

checking…

CARE runs on Docker. Install Docker Desktop, start it, then Check.

2 · Git required

checking…

Used once to download + build CARE. Install Git (Git for Windows / Xcode CLT / apt-get git), then Check.

3 · Network name — care.local required

checking…

Devices reach the clinic at http://care.local. This must be set before installing.

4 · Install location

Default app location

Where CARE's files are installed. Patient data is stored in Docker regardless. Leave default if unsure.

5 · Backup location

Desktop (default)

Daily database + file backups go here. Pick a USB/external drive so a dead disk doesn't take the backups with it. Leave default for the Desktop.

6 · Admin password

Complete steps 1–3 to continue.

          Restore from a backup
          

            Restoring replaces the current database (and uploaded files, when the backup
            has them) with the one you pick. CARE is stopped during the restore and restarted after.
            This can't be undone — take a fresh Backup now first if unsure.

          ⚠ Uninstall CARE Clinic
          

            Removes CARE's containers and all data volumes (patient records and uploaded
            files), the installed files, and the downloaded source. This permanently deletes all
            patient data and can't be undone. Make sure you have a backup you can rely on first.
